# Bussard

<img src="http://p.hagelb.org/bussard.png" alt="screenshot" />

Spaceships, planets, orbits, etc. in Lua.

## Usage

Use `make` to launch.

* `up` to accelerate
* `left`/`right` to turn
* `+`/`-` to zoom
* `~` to enter the repl

Requires [LÖVE](http://love2d.org) 0.8.0.
